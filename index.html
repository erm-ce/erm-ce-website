Getting started

Up and running in minutes

ERM-CE is designed to be self-hosted with minimal setup. No paid plans. No feature gates.

STEP 01
Clone the repo
Fork or clone ERM-CE from GitHub. All source code is open and yours to modify.
STEP 02
Configure your .env
Set your Discord token, MongoDB URI, and optional API keys in the .env template provided.
STEP 03
Run the bot
Start with python main.py or use the provided shell script for production deployments.
STEP 04
Run /setup
Use the guided /setup wizard to configure your server, roles, channels, and features.
